// Why FenrirOS

Built Different

⚡
OpenRC Init

Fast, lightweight, and transparent. OpenRC boots your system without the complexity of systemd — and we've made it approachable for everyday gamers.

systemd-free
🎮
Gaming First

Steam, Proton, Lutris, MangoHud, and GameMode — all pre-configured and working out of the box. Full Linux gaming stack without compromise.

plug & play
🐺
CachyOS Kernel

Built on the linux-cachyos kernel with BORE and LAVD schedulers for lowest input latency and best frame timing on Linux.

performance
🔗
elogind Sessions

Full desktop session management without systemd-logind. elogind gives DEs, Wayland compositors, and Steam everything they need.

compatibility
🔊
Pipewire Audio

Modern low-latency audio via Pipewire — running cleanly on OpenRC without systemd socket activation. Gaming audio that just works.

low latency
🛡️
Arch Foundation

Rolling release, pacman, AUR access, CachyOS repos. All the power of Arch with a curated gaming-first experience on top.

rolling release

In Norse myth, Fenrir was bound by chains the gods feared to break. We broke ours.

FenrirOS exists because gaming Linux shouldn't require systemd. OpenRC is fast, transparent, and powerful — it just needed someone to make it accessible.

// Roadmap

The Path

  • v0.1 — Now
    Foundation
    Bootable ISO with OpenRC + elogind + KDE Plasma. Core archiso profile established.
  • v0.2
    Gaming Stack Validated
    Steam, Proton, GameMode all working cleanly on OpenRC. elogind patches complete.
  • v0.3
    Branded Installer
    Calamares installer with FenrirOS branding. GPU auto-detection. One-click setup.
  • v0.4
    FenrirOS Repo
    Custom package repository for patched packages. fenrir- prefixed tooling.
  • v1.0
    Stable Release
    Public stable release. DistroWatch listing. The wolf runs free.
